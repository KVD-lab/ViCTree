Installation

ViCTree can be installed on any UNIX/LINUX machine. This pipeline is tested and best works with Ubuntu machine with multiple cores.

Prerequisites

Following programs must be available in $PATH environment.

  • BioPerl
  • Clustal-Omega
  • GNU BASH environment
  • Legacy BLAST (blastall,formatdb)
  • RAxML-PThreads

Download the ViCTree from this page and make sure all scripts are available in $PATH.

Running ViCTree

To run ViCTree pipeline you will need to specify 2 prior parameters

  • NCBI taxID of virus family or sub-family (e.g. 40120)
  • Seed set in .fasta format - This file should contain all fasta protein sequences representing diversity of the virus family or sub-family of interest

Following command will launch ViCTree analysis pipeline for Densovirinae sub-family 

ICTV_pipeline.sh -t 40120 -s denso_seeds_all.fa -l 100 -c 50 -r -p 10

Output

When pipeline is run for the first time a folder with taxID name is created that saves all output files generated by the pipeline.

Major output files generated by the pipeline include

  • taxID.fa - All protein sequences downloaded from NCBI for specified taxID
  • taxID_final_set.fa - Final set of sequences used for Multiple Sequence Alignment
  • taxID_seq_info - GIs grouped together representing the exact same sequence
  • taxID_tree.nhx - Final tree generated and rerooted by RAxML
  • taxID_clustalo_dist_mat.csv - Pairwise distance matrix file
  • taxID_metadata - Metadata file for each sequence from taxID_final_set.fa. Columns represent: Protein_GI,Nucleotide_GI,Genome_Accession,Species_Name,Description,Lineage,Genus,URL
